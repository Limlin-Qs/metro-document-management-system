# Navigation Styles Reference

Available navigation styles for web slide presentations.

---

## Available Navigation Styles

| Style | Position | Appearance | Best For |
|-------|----------|------------|----------|
| `dots-right` | 右侧垂直 | 小圆点竖排 | 简约主题，不干扰内容 |
| `dots-bottom` | 底部水平 | 小圆点横排 | 友好风格，常见布局 |
| `arrows-bottom` | 底部水平 | 圆点 + 左右箭头按钮 | 科技感，明确导航 |
| `progress-only` | 顶部 | 仅进度条，无可点击导航 | 极简风格，沉浸体验 |
| `breadcrumbs` | 顶部或底部 | 数字面包屑 01 / 02 / 03 | 商务/编辑风格 |
| `minimal` | 无可见元素 | 仅键盘/滑动/滚轮 | 全屏沉浸式 |

---

## Theme-Navigation Matching (Defaults)

| Theme | Default Navigation | Reason |
|-------|-------------------|--------|
| **Bold Signal** | `arrows-bottom` | 高冲击力，明确引导 |
| **Neon Cyber** | `arrows-bottom` | 科技感，带发光效果 |
| **Terminal Green** | `progress-only` | 极简黑客风 |
| **Dark Botanical** | `dots-right` | 优雅不干扰 |
| **Swiss Modern** | `dots-right` | 精确简约 |
| **Pastel Geometry** | `dots-bottom` | 友好可爱 |
| **Paper & Ink** | `breadcrumbs` | 编辑/文学风格 |
| **Notebook Tabs** | `breadcrumbs` | 报告/清单风格 |
